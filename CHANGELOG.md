# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

### Added

- Initial implementation of the mcp-pcloud MCP server for interacting with the pCloud API.
- Core MCP tools covering the pCloud API surface.
- Documentation site with GitHub Pages deployment workflow and a release workflow.
- Logo and icon assets for the app and the docs site, including favicons.

### Changed

- Updated README branding with a hero logo.
- Enlarged and centred the hero logo on the docs site.

### Chore

- Moved token storage to `~/.config/pcloud` and removed the `pcloud-cli` dependency.
- Updated the logo to a blueprint style and regenerated all favicon assets.
- Removed `border-radius` from the docs hero image.
